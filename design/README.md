## 1. 当前项目基准

《命运的织线》是一款以土地公视角展开的 60 天小镇命运干预游戏。玩家扮演归潮镇土地公，被岛民称为“岛爷”，通过命运织线器观察居民缘线、业线和关键事件，并以有限神力进行托梦、赐福、牵引机缘、遮蔽灾兆等间接干预。

当前项目基准如下：

| 项目 | 当前定稿 |
|---|---|
| 引擎 | Godot |
| 美术方向 | 简约 2D 像素风 |
| 主舞台 | 归潮镇 |
| 玩家身份 | 土地公，岛民称“岛爷” |
| 游戏时长 | 60 天，早 / 午 / 晚三个时段 |
| 核心资源 | 香火、神力、阳德、阴德 |
| 核心循环 | 观察命运 → 选择干预 → 时间推进 → 事件结算 → 命运变化 |
| MVP 原则 | 规则驱动优先，AI 文本作为可控叙事包装 |

## 6. 当前一致性规则

后续所有策划文档应统一使用：

- 地点：归潮镇
- 玩家身份：土地公 / 岛爷
- 核心资源：香火、神力、阳德、阴德
- 时间结构：60 天，早 / 午 / 晚
- 角色名：林潮音、陈远舟、顾沉舟、江雪仪、许明川、陈海生、许晴、周行知、慧圆、苏婉、叶可可、林月琴、何老三、赵守正
- 美术方向：简约像素风
- 实现方向：Godot，低成本、本地数据优先

## 7. 内容尺度边界

本项目可以使用黑色幽默、神仙官僚讽刺、命运压力和民俗恐怖暗示，但应避免露骨性描写、直接性剥削表达和过度猎奇内容。地府、邪教、悲剧等内容应以暗示、后果和氛围表达为主。
