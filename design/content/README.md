## 1. 目录职责

`content/` 用于存放所有偏内容生产的策划文件，面向剧情设计、任务设计、UI 文案、数据配置和测试验收。

本目录回答的问题是：

- 谁会出现？
- 他们在什么时间、什么地点、做什么事？
- 玩家能在哪里干预？
- 事件会导向什么内容结果？
- 哪些文本是固定的，哪些文本允许 AI 润色？

## 6. 内容尺度边界

允许：

- 民俗神明幽默
- 官僚讽刺
- 小镇秘密
- 邪教和灾难的暗示
- 家庭矛盾和人物压力

避免：

- 露骨性描写
- 猎奇化苦难
- 直接鼓励自伤或伤害他人
- 把敏感内容当作笑点
- AI 自行扩写超出设定边界的暴力或色情内容

涉及黑暗内容时，优先用后果、氛围、隐喻和角色反应表达。
