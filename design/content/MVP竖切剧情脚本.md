# MVP 竖切剧情脚本

> 文档状态：初版  
> 更新日期：2026-06-13  
> 用途：为首个研发竖切提供可实现、可测试、可演示的最小剧情流程

---

## 1. 竖切目标

首个竖切目标是证明《命运的织线》的核心循环成立：

```text
主菜单 → 新游戏 → 开局剧情 → 主地图 HUD → 查看 3 名 NPC → 触发 1 个事件 → 使用 1 次干预 → 推动时间 → 显示结算
```

竖切不追求完整 60 天体验，只需要完成一个可玩的短流程。

---

## 2. 竖切范围

### 2.1 包含内容

| 模块 | 内容 |
|---|---|
| 开局 | 天界使者送命运织线器 |
| 地图 | 寺庙、学校、咖啡店、海鲜店 |
| NPC | 林潮音、陈远舟、陈海生 |
| 事件 | 咖啡店复习、陈海生发脾气 |
| 干预 | 托梦、赐福二选一 |
| 资源 | 香火、神力、阳德、阴德 |
| 结算 | 单次事件结果 + 当日时段推进摘要 |

### 2.2 不包含内容

- 完整 60 天内容。
- 所有 NPC。
- 完整结局系统演出。
- 在线 AI 文本生成。
- 复杂关系网可视化。
- 多周阶段事件池。

## 3. 竖切角色

### 3.1 土地公 / 岛爷

玩家身份。竖切中只通过旁白、系统提示和 UI 操作体现，不需要立绘。

### 3.2 林潮音

| 项目 | 内容 |
|---|---|
| 初始地点 | 学校 / 咖啡店 |
| 初始状态 | 噩梦频繁、压力轻微 |
| 竖切作用 | 展示人物面板、缘线和日常事件 |

### 3.3 陈远舟

| 项目 | 内容 |
|---|---|
| 初始地点 | 学校 / 咖啡店 |
| 初始状态 | 备考中、想离岛 |
| 竖切作用 | 与林潮音形成第一条可见缘线 |

### 3.4 陈海生

| 项目 | 内容 |
|---|---|
| 初始地点 | 海鲜店 |
| 初始状态 | 压力高、怨气积累 |
| 竖切作用 | 展示高风险事件和干预反馈 |

## 4. 剧情流程

### 4.1 主菜单

玩家点击：

```text
新游戏
```

进入开局剧情。

### 4.2 开局剧情节选

场景：山顶旧土地庙，夜。

```text
夜风从神龛缝里吹进来，香炉里的灰已经冷了。
```

```text
岛爷正趴在供桌旁打瞌睡，功德簿摊开在一边。
```

天界使者：

```text
归潮镇土地公，接天界监察司新设备。
```

```text
这是命运织线器。
```

```text
六十天内，观察岛民命运，维持香火，阻止归潮镇彻底滑向灾祸。
```

系统提示：

```text
命运织线器已启动。
剩余时间：60 天。
当前时段：Day 1 早。
```

进入主地图 HUD。

## 6. 人物查看流程

玩家需要至少查看 3 名 NPC。

### 6.1 林潮音人物摘要

```text
林潮音
高中生。最近常做关于潮水的噩梦。
```

可见状态：

```text
压力：轻微
命运征兆：潮声入梦
```

### 6.2 陈远舟人物摘要

```text
陈远舟
高中生。正在准备离开归潮镇的考试。
```

可见状态：

```text
目标：升学离岛
关系：与林潮音互相支持
```

### 6.3 陈海生人物摘要

```text
陈海生
渔民，海鲜店老板。最近脾气越来越差。
```

可见状态：

```text
压力：明显
风险：家庭冲突
```

## 7. 事件一：咖啡店复习

### 7.1 触发条件

```text
EventId: w1_chaoyin_study
Day: 1
TimeSlot: Afternoon
Location: coffee_shop
Participants: lin_chaoyin;chen_yuanzhou
```

### 7.2 事件正文

```text
咖啡店的窗边，林潮音把练习册翻到最后一页，又偷偷合上。
```

陈远舟：

```text
你昨晚又没睡好？
```

林潮音：

```text
只是梦见海水涨得很高。
```

陈远舟：

```text
等考完，我们就离这里远一点。
```

### 7.3 玩家选项

| 选项 | 效果 |
|---|---|
| 观察 | 不消耗神力，记录事件 |
| 赐福 | 消耗 3 神力，降低两人压力，增加阳德 |

### 7.4 结算文本

未干预：

```text
他们安静复习，像归潮镇里许多个普通下午一样。
```

赐福：

```text
窗外的风轻了一点，两个人都暂时相信未来还来得及。
```

## 8. 事件二：陈海生发脾气

### 8.1 触发条件

```text
EventId: w3_chenhai_tantrums
竖切中可提前模拟到 Day 1 Night
Location: seafood_shop
Participants: chen_haisheng;chen_yuanzhou
```

### 8.2 事件正文

```text
海鲜店已经打烊，门口的水桶还没收。
```

```text
陈海生把账本摔在桌上，声音惊得陈远舟停在门口。
```

陈海生：

```text
读书读书，读出去以后，这个家谁管？
```

陈远舟：

```text
我不是要丢下你们。
```

### 8.3 玩家选项

| 选项 | 效果 |
|---|---|
| 观察 | 家庭关系下降，陈海生压力上升 |
| 托梦 | 消耗 2 神力，让陈海生短暂迟疑 |
| 赐福 | 消耗 3 神力，降低当次冲突伤害 |

### 8.4 结算文本

观察：

```text
陈家的争吵像潮湿的木柴，越堆越高。
```

托梦：

```text
陈海生忽然想起一个关于翻船的梦，话到嘴边停了一下。
```

赐福：

```text
这次争吵被压低了声音，但问题还在屋里。
```

## 9. 时间推进结算

玩家点击：

```text
推进时辰
```

系统显示：

```text
Day 1 晚 → Day 2 早
```

结算摘要：

```text
本时段命运变化：
- 林潮音与陈远舟的缘线略有变化。
- 陈海生的家庭压力仍在累积。
- 神力因干预减少。
```

若玩家使用赐福或托梦：

```text
阳德 +2
神力 -2 / -3
```

---

## 10. 竖切完成条件

竖切演示完成时，玩家应完成：

- [ ] 从主菜单进入新游戏。
- [ ] 看完开局剧情。
- [ ] 进入主地图 HUD。
- [ ] 查看林潮音、陈远舟、陈海生人物面板。
- [ ] 触发至少 1 个事件。
- [ ] 使用至少 1 次神力干预。
- [ ] 看见资源变化。
- [ ] 推进一次时辰。
- [ ] 看见结算摘要。

---

## 11. 研发备注

- 所有文本可先使用本地模板，不接 AI。
- 事件结算使用 `事件结果表.csv` 中的结构化结果。
- UI 只展示状态，不拥有资源和 NPC 数据。
- 竖切可以使用固定触发顺序，不必先实现完整事件权重系统。
- 完整随机事件池可放到下一阶段。
