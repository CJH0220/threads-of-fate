# 《命运的织线》开局固定剧情

> 类型：固定剧情 / 新手引导  
> 触发时机：新游戏开始  
> 玩家权限：只能点击推进，关键剧情不可干预  
> 时间框架：第0夜（天界/hell使者送设备） → Day 1-60（2026年5月18日-7月17日）  
> 文件版本：v1.0 整合版本

---

## 1. 概念设定

### 1.1 玩家身份

- 土地公（岛上居民称“岛爷”）
- 负责归潮镇的一方水土
- 当前状态：懒散、香火惨淡、处于被撤职风险
- 核心困境：现代科技发展，信神信佛的人越来越少，香火连年下降

### 1.2 天界界面设定

**设备概念**：天界使者送来一台 **「命运织线器」**（外形类似 iPad，但材质如玉），这是本游戏的主界面。

**功能**：
- 将人与人之间的缘线可视化
- 将每个人的命运节点和走向呈现
- 用于观察、记录、有限的干预
- 是天界的新式管理系统试点项目

**为什么归潮镇？**
- 岛小、人员少，适合测试新系统
- 即将发生大难，需要试点数据
- 反正现在信众少，倒掉也不会太丢人

### 1.3 时间框架

- **第0夜**：天界/hell使者送来设备，初次觉醒
- **游戏时长**：2026年5月18日-7月17日（60天，约2个月）
- **第60天**：天灾降临，归潮镇命运分晓

## 4. 第0夜：第一次可操控事件

**这是开局唯一允许玩家进行轻微干预的事件**，用于教学。

### 4.1 事件：夜行回家的邀约

**时间**：第0夜，晚  
**地点**：放学路上  
**参与人物**：陈远舟、林潮音  
**类型**：日常事件，轻微影响命运走向

**事件描述**：

```
夜风轻拂，路灯拉长。

陈远舟和林潮音并肩走在放学路上。

两人都沉默着，但沉默里有些别的什么。
```

```
陈远舟的手在口袋里，手指碰到了一张折叠的纸——

那是5月20日的游乐园门票，早早就买好的。

他看了林潮音一眼，又移开目光，喉咙动了动。
```

---

### 4.2 缘线状态显示

**当前缘线状态**：
- 林潮音 - 陈远舟：爱情线强度3级
- 陈远舟魅力值：5级
- 林潮音对远舟好感度：6级

---

### 4.3 玩家可操控选项

**注意**：这是开局教学事件，只能轻度调整缘线，不能直接改变对话走向。

#### 选项1：不干预（默认）

**结果**：
- 不触发邀约
- 缘线保持现状
- 系统提示：缘分自有天定

#### 选项2：调整缘线+1级

**效果**：
- 爱情线强度：3 → 4级
- 触发条件：玩家使用一次神力（赐福）

**结果描述**：

```
你轻轻拨动了一根丝线。

陈远舟忽然觉得有什么东西推了他一把。
```

```
他停住脚步。
```

```
「潮音，」他说，「5月20号，游乐园，要不要一起去？」
```

```
林潮音抬起头，路灯的光落在她脸上。

她犹豫了片刻，然后轻轻点头。

「嗯，」她说。
```

```
两人继续走，但步子轻了一些。
```

**系统提示**：

```
缘线已调整。
```

```
爱情线强度提升至4级。
```

```
下次关键事件判定将受此影响。
```

---

#### 选项3：赐福陈远舟的勇气

**效果**：
- 消耗神力
- 提高邀约成功率（命运硬币环节）
- 需要进行命运硬币判定

**命运硬币环节**：

```
陈远舟勇气值 + 缘线强度 = 基础硬币数
```

```
你使用赐福，额外增加1枚硬币
```

```
需求值：3枚硬币成功
```

**分支1：成功（3枚以上）**

```
陈远舟深吸一口气，像是得到了谁的鼓励。

「潮音，」他转头看她，「5月20号，游乐园。」

「我想约你一起去。」

林潮音眼睛亮了。

「好，」她说，「我去。」
```

```
爱情线强度：4 → 5级
```

**分支2：失败（不足3枚）**

```
陈远舟犹豫了。

他想起家里的压力，想起母亲苏婉最近总是悄悄叹气。

「潮音，」他最后说，「你最近学习压力大吗？」

「有空的话，我们可以一起复习功课。」

林潮音的眼神黯淡了一些。

「嗯，好，」她说。
```

```
爱情线强度保持3级
```

## 7. 开局剧情的设计取舍
