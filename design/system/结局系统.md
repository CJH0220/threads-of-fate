# 结局系统

> 文档状态：初版  
> 更新日期：2026-06-12  
> 所属阶段：P0  
> 用途：定义 Game Over、60 天总评、路线结局、角色结局和结局记录

---

## 1. 系统目标

结局系统负责把 60 天内的资源、事件、NPC 命运和玩家路线倾向汇总为最终结果。

系统目标：

1. 给 60 天游戏一个明确收束。
2. 让玩家理解自己为什么得到这个结局。
3. 支持正神、邪神、中性、传奇和失败路线。
4. 将关键 NPC 的命运纳入总评。
5. 支持结局回顾和后续扩展。

---

## 2. 结局层级

结局系统分三层：

| 层级 | 内容 | 示例 |
|---|---|---|
| 全局路线 | 玩家作为土地公的最终路线 | 正神 / 邪神 / 中性 / 传奇 / 失败 |
| 小镇状态 | 归潮镇整体命运 | 庙会保住、邪教瓦解、香火熄灭 |
| 角色结局 | 关键 NPC 的个人命运 | 林潮音留下、陈远舟离岛、陈海生救赎 |

---

## 3. 结局类型

### 3.1 失败 / Game Over

失败代表玩家未能维持神位或小镇出现不可逆崩坏。

触发来源：

- 香火低于失败阈值。
- 强硬不幸结局触发。
- 邪教完全失控且玩家无法收束。
- 关键灾难造成不可恢复后果。

### 3.2 正神路线

玩家主要通过守护、引导、救助积累阳德，并保持香火稳定。

表现：

- 土地公神位保住。
- 小镇记住旧神。
- 多数关键 NPC 得到较好或可接受结局。
- UI 使用金色、暖光、香火旺盛表现。

### 3.3 邪神路线

玩家通过操控、遮蔽、牺牲或地府路线积累阴德，并以另一种方式获得力量。

表现：

- 香火可能仍高，但信仰性质扭曲。
- 小镇畏惧或依赖土地公。
- 某些 NPC 成为代价。
- UI 使用紫黑、暗红、压抑小镇表现。

### 3.4 中性路线

玩家勉强完成考核，但没有显著走向守护或邪路。

表现：

- 神位暂时保住。
- 小镇继续生活，但许多问题没有真正解决。
- 角色结局混杂。
- UI 使用暖灰、平淡延续表现。

### 3.5 传奇路线

玩家在香火、阳德、关键事件和 NPC 结局上都取得突出结果。

表现：

- 土地公成为归潮镇真正的守护神。
- 小镇在天灾后形成新的信仰传说。
- 多数关键 NPC 命运改善。
- UI 使用金紫混合、传说化光效表现。

## 4. 判定时机

### 4.1 过程判定

游戏过程中可随时检查：

- 是否触发 Game Over。
- 是否出现强硬不幸结局。
- 是否锁定某些角色结局。
- 是否开启特殊路线条件。

### 4.2 最终判定

最终路线在 Day 60 晚结算后判定。

判定顺序：

```text
1. 检查强硬 Game Over
2. 检查香火失败
3. 计算关键 NPC 结局
4. 计算小镇状态
5. 计算阳德 / 阴德路线倾向
6. 检查传奇路线
7. 输出最终路线
8. 生成 60 天总评
9. 进入结局演出或文本结局
```

## 5. 全局路线判定

### 5.1 推荐默认条件

| RouteId | 路线 | 推荐条件 |
|---|---|---|
| `failure` | 失败 | 香火 < 100，或触发强硬不幸结局 |
| `righteous_god` | 正神 | 香火 ≥ 100，阳德 ≥ 阴德 + 20 |
| `evil_god` | 邪神 | 香火 ≥ 100，阴德 ≥ 阳德 + 20 |
| `neutral_god` | 中性 | 香火 ≥ 100，阳德与阴德差距 < 20 |
| `legend_god` | 传奇 | 香火 ≥ 150，阳德 ≥ 80，关键 NPC 好结局 ≥ 4，邪教未成功 |

### 5.2 判定优先级

| 优先级 | 判定 |
|---:|---|
| 1 | 强硬不幸结局 |
| 2 | 香火失败 |
| 3 | 传奇路线 |
| 4 | 邪神路线 |
| 5 | 正神路线 |
| 6 | 中性路线 |

传奇路线高于正神路线，但不高于失败。

## 6. Game Over 条件

### 6.1 香火失败

| 条件 | 结果 |
|---|---|
| Day 60 晚，香火 < 100 | 香火熄灭 Game Over |
| 游戏中香火降到 0 且持续一整天 | 可触发提前失败，P1 |

### 6.2 强硬不幸结局

| 条件 | 结果 |
|---|---|
| 寺庙被烧毁且无补救 | 神位崩塌 |
| 邪教完成换神仪式 | 归潮镇沦陷 |
| 集体叛教达到不可逆阈值 | 香火断绝 |
| 关键天灾应对完全失败 | 小镇重大伤亡 |

P0 可先实现 Day 60 香火失败和一个强硬不幸结局。

## 7. 角色结局判定

### 7.1 角色结局类型

| 类型 | 说明 |
|---|---|
| Good | 明显改善，角色获得较好未来 |
| Neutral | 问题未完全解决，但没有崩坏 |
| Bad | 角色命运恶化或付出沉重代价 |
| Unknown | 玩家未充分接触，结局模糊 |

### 7.2 MVP 角色结局

| NPC | Good | Neutral | Bad |
|---|---|---|---|
| 林潮音 | 自愿选择守护或自由离岛，精神稳定 | 留在小镇但仍受束缚 | 被利用为巫女或精神崩溃暗示 |
| 陈远舟 | 成功离岛或与潮音达成健康约定 | 留岛承担家庭责任 | 梦想破灭或被邪教利用 |
| 陈海生 | 承认错误并修复家庭 | 家庭关系勉强维持 | 投向邪教、纵火或家庭破裂 |
| 顾沉舟 | 被揭露并驱逐或逮捕 | 逃离但邪教受挫 | 完成换神或严重腐化小镇 |
| 慧圆 | 腐败被揭露并承担后果 | 寺庙改革不彻底 | 腐败引发信仰崩塌或事故 |

## 8. 小镇状态判定

| 状态 | 条件示例 |
|---|---|
| 信仰恢复 | 香火 ≥ 120，寺庙危机解决 |
| 勉强延续 | 香火 ≥ 100，但关键问题未解决 |
| 邪教阴影 | 邪教传播值高，顾沉舟未被阻止 |
| 灾后重建 | 天灾造成损失，但核心 NPC 多数存活 |
| 归潮镇沦陷 | 邪教成功或强硬不幸结局 |

小镇状态用于 60 天总评和结局文本，不一定单独决定路线。

总评界面至少显示：

- 最终路线
- 香火 / 阳德 / 阴德 / 神力最终值

- 小镇状态总结
- 查看最终结局按钮

## 13. 边界情况

| 情况 | 处理 |
|---|---|
| 同时满足传奇和失败 | 失败优先 |
| 同时满足正神和传奇 | 传奇优先 |
| 阳德阴德差距很小 | 中性路线 |
| 关键 NPC 未解锁 | 显示 Unknown 或“命运未明” |
| 角色结局与全局路线冲突 | 允许，作为复杂后果展示 |
| 玩家触发邪神路线但阻止邪教 | 可进入地府 / 阴德型邪神路线 |

## 15. 验收标准

- [ ] Day 60 晚能触发最终判定。
- [ ] 香火 < 100 时进入 Game Over。
- [ ] 阳德高于阴德时进入正神路线。
- [ ] 阴德高于阳德时进入邪神路线。
- [ ] 差距不明显时进入中性路线。
- [ ] 满足高香火、高阳德、多好结局时进入传奇路线。
- [ ] 60 天总评能显示最终资源和 NPC 结局。
- [ ] Game Over 能显示失败原因和触发时间。
- [ ] 结局记录可被保存或预留接口。
