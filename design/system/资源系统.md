# 资源系统

> 文档状态：初版  
> 更新日期：2026-06-12  
> 所属阶段：P0  
> 用途：定义香火、神力、阳德、阴德的规则、来源、消耗、阈值与结局影响

---

## 1. 系统目标

资源系统负责把玩家的神明行为转化为可见、可调、可结算的数值压力。资源不是单纯货币，而是玩家路线和小镇状态的核心反馈。

系统目标：

1. 给玩家明确的短期行动成本。
2. 给 60 天终局提供路线判定依据。
3. 让正向守护、阴暗操控、放任失败都能被系统记录。
4. 为 UI、事件系统、干预系统、结局系统提供统一数值来源。

---

## 2. 核心资源

| 资源 | 定位 | 玩家理解 |
|---|---|---|
| 香火 | 神位稳定与表层胜负 | 岛民还记不记得我、供不供我 |
| 神力 | 行动消耗资源 | 我还能干预多少次 |
| 阳德 | 正向行为积累 | 我是否守护、引导、救助了他们 |
| 阴德 | 负向行为积累 | 我是否操控、遮蔽、利用了他们 |

---

## 3. 初始值与范围

| 资源 | 初始值 | 最小值 | 最大值 | 说明 |
|---|---:|---:|---:|---|
| 香火 | 50 | 0 | 999 | Day 60 低于阈值可能失败 |
| 神力 | 15 | 0 | 99 | 用于观察强化和神力干预 |
| 阳德 | 0 | 0 | 999 | 正向路线判定 |
| 阴德 | 0 | 0 | 999 | 邪路 / 操控路线判定 |

MVP 阶段最大值可以只用于 UI 裁剪，不必做复杂溢出逻辑。

---

## 4. 玩家可见信息

### 4.1 顶栏显示

主地图 HUD 顶栏显示：

```text
香火 50    神力 15    阳德 0    阴德 0
```

### 4.2 变化反馈

资源变化时必须显示变化原因：

```text
香火 +3：村民感谢土地公庇佑
神力 -2：使用托梦
阳德 +5：阻止陈海生冲突升级
阴德 +3：遮蔽许明川获得证据
```

### 4.3 结算显示

时间推进、事件结算、周总结、60 天总评都应显示关键资源变化。

## 5. 香火规则

### 5.1 香火来源

| 来源 | 示例 | 默认变化 |
|---|---|---:|
| 正向事件成功 | 阻止事故、解决家庭冲突 | +3 到 +10 |
| 庙会 / 信仰活动顺利 | 寺庙恢复公信力 | +5 到 +15 |
| NPC 好结局 | 角色命运改善 | +5 到 +20 |
| 周阶段表现良好 | 本周无重大失败 | +3 到 +8 |

### 5.2 香火损失

| 来源 | 示例 | 默认变化 |
|---|---|---:|
| 危险事件失败 | 火灾、事故、失踪 | -5 到 -20 |
| 邪教传播扩大 | 新信仰集会成功 | -5 到 -15 |
| 寺庙腐败暴露未处理 | 信任崩塌 | -5 到 -12 |
| 玩家长期不干预 | 小镇命运恶化 | -1 到 -3 / 周 |

### 5.3 香火阈值

| 阈值 | 状态 | 表现 |
|---:|---|---|
| 0-29 | 神位崩塌 | 高危，可能提前 Game Over |
| 30-59 | 香火惨淡 | UI 提示压力 |
| 60-99 | 勉强维持 | 中性区间 |
| 100-149 | 神位稳定 | 可进入正常结局 |
| 150+ | 香火鼎盛 | 可支持传奇路线条件 |

## 6. 神力规则

### 6.1 神力来源

| 来源 | 默认变化 |
|---|---:|
| 每日开始自然恢复 | +1 |
| 周总结奖励 | +3 到 +8 |
| 香火高时额外恢复 | +1 到 +3 |
| 特定事件奖励 | +1 到 +5 |

### 6.2 神力消耗

| 行为 | 默认消耗 |
|---|---:|
| 托梦 | 2 |
| 赐福 | 3 |
| 牵引机缘 | 4 |
| 遮蔽灾兆 | 3 |
| 显灵警示 | 5 |
| 强化观察 | 1 |

### 6.3 神力不足

神力不足时：

- 禁用对应干预按钮。
- 显示消耗与当前神力。
- 点击后显示 Toast：

```text
神力不足，无法执行该干预。
```

## 7. 阳德规则

阳德代表玩家以守护、引导、救助为主的路线。

### 7.1 阳德来源

| 行为 | 默认变化 |
|---|---:|
| 避免事故 | +5 到 +15 |
| 帮助 NPC 面对真相 | +3 到 +10 |
| 促成和解 | +3 到 +8 |
| 阻止邪教伤害 | +5 到 +15 |
| 保护弱势角色 | +3 到 +10 |

### 7.2 阳德用途

阳德不作为普通消耗资源，主要用于：

- 正神路线判定
- 传奇路线条件
- 某些高阶正向干预解锁
- 60 天总评展示

---

## 8. 阴德规则

阴德代表玩家以操控、遮蔽、诱导、牺牲为主的路线。

### 8.1 阴德来源

| 行为 | 默认变化 |
|---|---:|
| 遮蔽关键证据 | +2 到 +8 |
| 引导 NPC 做出伤害性选择 | +5 到 +15 |
| 放任邪教扩大但换取短期收益 | +5 到 +20 |
| 利用他人弱点达成目标 | +3 到 +10 |
| 为维持香火牺牲少数人 | +10 到 +25 |

### 8.2 阴德用途

阴德主要用于：

- 邪神路线判定
- 阴德技能解锁
- 结局文案风格
- 地府路线事件触发

### 8.3 阴德不是失败值

阴德高不等于直接失败。阴德高可能导向邪神路线或黑暗胜利；失败通常由香火崩塌、关键灾难失控、邪教不可控等条件触发。

---

## 9. 资源变化结算顺序

事件结算时按以下顺序处理：

```text
1. 事件基础结果
2. 玩家干预修正
3. NPC 状态变化
4. 香火变化
5. 神力消耗或奖励
6. 阳德 / 阴德变化
7. 路线倾向更新
8. UI 变化摘要
```

## 10. 与结局系统的关系

### 10.1 基础路线倾向

| 条件 | 倾向 |
|---|---|
| 香火 < 100 且无特殊补救 | 失败风险 |
| 阳德明显高于阴德 | 正神路线 |
| 阴德明显高于阳德 | 邪神路线 |
| 阳德和阴德都低 | 中性 / 平淡延续 |
| 香火高、阳德高、关键 NPC 多数好结局 | 传奇路线 |
| 香火高、阴德高、邪教或地府路线成功 | 邪神强势路线 |

### 10.2 推荐判定区间

| 路线 | 推荐条件 |
|---|---|
| 失败 | 香火 < 100，或触发强硬不幸结局 |
| 正神 | 香火 ≥ 100，阳德 ≥ 阴德 + 20 |
| 邪神 | 香火 ≥ 100，阴德 ≥ 阳德 + 20 |
| 中性 | 香火 ≥ 100，阳德与阴德差距 < 20 |
| 传奇 | 香火 ≥ 150，阳德 ≥ 80，关键 NPC 好结局 ≥ 4 |

具体判定由 `结局系统.md` 定义。

## 13. 边界情况

| 情况 | 处理 |
|---|---|
| 神力不足 | 禁用技能并提示 |
| 香火降到 0 | 标记高危，可触发 Game Over 检查 |
| 资源变化过多 | 时间推进面板只显示高优先级变化 |
| 同一事件同时加阳德和阴德 | 允许，表示复杂代价 |
| 资源超过最大值 | UI 可显示最大值或实际值，按实现决定 |
| 资源低于 0 | 裁剪为 0 |

## 14. 调参项

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `InitialIncense` | 50 | 初始香火 |
| `InitialDivinePower` | 15 | 初始神力 |
| `DailyDivinePowerRegen` | 1 | 每日神力恢复 |
| `FailureIncenseThreshold` | 100 | Day 60 基础失败阈值 |
| `LegendIncenseThreshold` | 150 | 传奇路线香火建议阈值 |
| `RouteDifferenceThreshold` | 20 | 阳德 / 阴德路线差距阈值 |

## 15. 验收标准

- [ ] UI 顶栏能显示四项核心资源。
- [ ] 使用干预会正确扣除神力。
- [ ] 神力不足时不能执行干预。
- [ ] 事件结算能改变香火、阳德或阴德。
- [ ] 时间推进摘要能显示资源变化原因。
- [ ] 香火低于阈值能被结局系统读取。
- [ ] 阳德 / 阴德差距能影响最终路线。
- [ ] 资源不会显示负数。
