# 音频方向

> 文档状态：初版  
> 更新日期：2026-06-13  
> 用途：定义《命运的织线》的音乐、环境音、音效和 UI 声音方向

---

## 1. 总体声音目标

音频应服务于低成本叙事模拟，不追求大量曲目，而追求稳定氛围。

关键词：

```text
海风 / 旧庙 / 香火 / 小镇日常 / 民俗不安 / 温暖像素 / 潮声预兆
```

---

## 2. 音乐方向

| 场景 | 音乐方向 | 优先级 |
|---|---|---|
| 主菜单 | 温暖、略带潮声和钟声 | P0 |
| 主地图白天 | 轻松小镇日常，木吉他或简单合成器 | P0 |
| 主地图夜晚 | 更空、更慢，加入海风和低频 | P0 |
| 事件日常 | 短循环，温暖克制 | P0 |
| 高风险事件 | 低频、鼓点稀疏、避免恐怖片化 | P0 |
| 周总结 | 稍有仪式感，像翻账本和潮声回响 | P1 |
| 终局 | 海潮、钟声、低频持续音 | P1 |
| 结局 | 根据路线切换明亮、压抑、空白或传奇感 | P1 |

P0 可用 4-5 首循环音乐完成：

1. 主菜单 / 寺庙主题。
2. 白天小镇主题。
3. 夜晚不安主题。
4. 日常事件主题。
5. 高风险事件主题。

---

## 3. 环境音

| 地点 | 环境音 |
|---|---|
| 寺庙 | 风铃、香灰、远处海风、偶尔钟声 |
| 学校 | 远处学生声、粉笔、窗外风 |
| 咖啡店 | 杯盘轻响、门铃、低声交谈 |
| 海鲜店 | 水桶、鱼箱、卷帘门、远处船声 |
| 港口 | 海浪、缆绳、船体摩擦、雾号 |
| 诊所 | 药柜、钟表、纸张、消毒水氛围 |
| 酒屋 | 杯声、低语、旧电扇 |
| 山林 | 虫鸣、树叶、远处不明低语 |

环境音应低存在感，不干扰阅读。

---

## 4. UI 音效

| 操作 | 音效方向 |
|---|---|
| 按钮确认 | 轻木鱼 / 轻敲玉片 |
| 返回 | 轻纸页翻动 |
| 打开命运织线器 | 玉石亮起 + 轻微电子声 |
| 事件出现 | 命运线轻拨声 |
| 高风险提示 | 低频短响 + 风声 |
| 资源增加 | 柔和上扬音 |
| 资源减少 | 轻微下坠音 |
| 保存成功 | 香火亮起的短音 |
| 错误 / 禁止 | 干涩木响，避免刺耳 |

---

## 5. 角色与事件声音

P0 不要求角色配音。

可选低成本表现：

- 对话文字出现时使用轻微文本音。
- 不同角色可使用不同音高或材质的文字音。
- 高风险事件降低文本音频率或加入风声。

禁止：

- 大量 AI 配音作为 MVP 必需项。
- 过度惊吓音效。
- 长时间高频噪声。

---

## 6. 路线声音差异

| 路线 | 声音倾向 |
|---|---|
| 正神 | 暖钟声、稳定和弦、海风变柔 |
| 邪神 | 低频、反向铃声、干燥木响 |
| 中性 | 空旷、留白、弱旋律 |
| 传奇 | 更完整的主旋律，加入人声哼唱或钟鼓 |
| Game Over | 声音抽空，留下风声和香火熄灭声 |

---

## 7. 音量与可访问性

设置界面至少支持：

- 主音量。
- 音乐音量。
- 音效音量。
- 环境音音量。
- 降低刺耳提示音。
- 文字速度不与音效强绑定。

关键危险提示不能只依赖声音，必须同时有视觉提示。

## 9. 验收标准

- [ ] P0 流程有完整音乐覆盖。
- [ ] UI 操作有基础反馈音。
- [ ] 高风险提示同时有视觉和声音反馈。

- [ ] 音频不依赖配音即可完成叙事。
