# UI 规格：地点面板

> 文档状态：初稿  
> 更新日期：2026-06-10  
> 所属阶段：P0  
> 引擎方向：Godot  
> 美术方向：简约像素风

---

## 1. 用途

地点面板用于查看归潮镇某个地点的简介、场景图、今日人物行程和相关事件。它帮助玩家判断某个地点在当前时段是否值得关注。

---

## 2. 入口与出口

### 入口

| 来源 | 触发 |
|---|---|
| 主地图 | 点击地点按钮 |
| 人物面板 | 点击行程地点 |
| 事件面板 | 点击事件发生地点 |

### 出口

| 操作 | 结果 |
|---|---|
| 关闭按钮 | 返回主地图或上一层 |
| Esc | 返回上一层 |
| 点击人物 | 打开人物面板 |
| 点击事件 | 打开事件详情 / 事件推进面板 |

---

## 3. 布局规格

```text
┌──────────────────────────────────────────────┐
│ 地点名：学校                         [关闭]  │
├──────────────────────┬───────────────────────┤
│ 地点简介              │ 像素地点图             │
│ 当前状态 / 特殊提示    │                       │
├──────────────────────┴───────────────────────┤
│ 今日出行                                      │
│ 早：林潮音、陈远舟     事件：课堂争执          │
│ 午：顾沉舟             事件：无                │
│ 晚：无人               事件：神秘脚印          │
└──────────────────────────────────────────────┘
```

---

## 4. 组件清单

| 组件 | 类型 | 内容 | 交互 |
|---|---|---|---|
| 地点名 | `Label` | 地点名称 | 无 |
| 地点图 | `TextureRect` | 像素场景图 | 无 |
| 简介区 | `RichTextLabel` | 地点介绍 | 可滚动 |
| 当前状态 | `Label` / 图标 | 特殊状态、天气影响 | 无 |
| 今日出行列表 | `VBoxContainer` | 早/午/晚人物和事件 | 人物、事件可点击 |
| 关闭按钮 | `Button` | 返回 | 关闭面板 |

---

## 5. 今日出行展示

| 时段 | 人物 | 事件 |
|---|---|---|
| 早 | 林潮音、陈远舟 | 课堂争执 |
| 午 | 顾沉舟 | 无 |
| 晚 | 无人 | 神秘脚印 |

规则：

- 人物名可点击打开人物面板。
- 已知事件可点击进入事件面板。
- 未解锁事件只显示模糊提示，如“似乎会发生什么”。
- 无人物时显示“无人停留”。
- 无事件时显示“无事件”。

---

## 6. 地点状态

| 状态 | 表现 |
|---|---|
| 普通 | 正常简介与今日出行 |
| 特殊日 | 显示特殊标签，如“庙会筹备中” |
| 危险 | 红色警告边框或图标 |
| 未解锁 | 地点按钮灰化，面板不可打开或只显示“尚未发现” |
| 暂不可用 | 显示原因，如“夜晚无法进入” |

---

## 7. Godot 实现建议

```text
LocationPanel (PanelContainer)
├── Header (HBoxContainer)
│   ├── LocationNameLabel
│   └── CloseButton
├── TopContent (HBoxContainer)
│   ├── DescriptionBlock (VBoxContainer)
│   └── LocationImage (TextureRect)
└── ScheduleBlock (VBoxContainer)
    ├── MorningRow
    ├── AfternoonRow
    └── NightRow
```

---

## 8. 数据需求

| 数据 | 来源 | 用途 |
|---|---|---|
| 地点基础信息 | LocationDatabase | 名称、简介、图片 |
| 地点状态 | LocationStateSystem | 特殊状态、危险状态 |
| 今日人物 | NPCScheduleSystem | 各时段人物列表 |
| 今日事件 | EventScheduler | 各时段事件列表 |
| 解锁状态 | WorldProgressSystem | 控制地点是否可见 |

---

## 9. 验收标准

- [ ] 点击主地图地点后能打开对应地点面板。
- [ ] 地点面板显示地点名、简介和像素地点图。
- [ ] 今日早/午/晚人物列表显示正确。
- [ ] 人物名可点击并打开人物面板。
- [ ] 事件名可点击并进入事件面板或事件详情。
- [ ] 无人物、无事件、未解锁地点都有明确状态文案。
- [ ] 危险地点或特殊日地点有明显视觉提示。
