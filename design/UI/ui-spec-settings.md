# UI 规格：设置界面

> 文档状态：初稿  
> 更新日期：2026-06-10  
> 所属阶段：P0  
> 引擎方向：Godot  
> 美术方向：简约像素风

---

## 1. 用途

设置界面用于调整音频、显示、文本速度、操作提示和辅助功能。它需要可从主菜单和游戏内暂停菜单进入。

---

## 2. 分类结构

```text
设置
├── 音频
├── 显示
├── 文本
├── 操作
└── 辅助功能
```

---

## 3. 布局规格

```text
┌──────────────────────────────────────┐
│ 设置                         [关闭]  │
├───────────┬──────────────────────────┤
│ 音频       │ 主音量      [-----|---]   │
│ 显示       │ 音乐        [---|-----]   │
│ 文本       │ 音效        [------|--]   │
│ 操作       │ 环境音      [----|----]   │
│ 辅助功能   │                          │
├───────────┴──────────────────────────┤
│ [恢复默认]              [应用] [返回] │
└──────────────────────────────────────┘
```

---

## 4. 设置项

### 4.1 音频

| 项 | 类型 | 默认 |
|---|---|---|
| 主音量 | Slider | 80% |
| 音乐 | Slider | 70% |
| 音效 | Slider | 80% |
| 环境音 | Slider | 60% |

### 4.2 显示

| 项 | 类型 | 默认 |
|---|---|---|
| 窗口模式 | OptionButton | 窗口化 |
| 分辨率 | OptionButton | 当前屏幕推荐 |
| 像素缩放 | OptionButton | 2x 或自动 |
| 屏幕震动 | Toggle | 开 |

### 4.3 文本

| 项 | 类型 | 默认 |
|---|---|---|
| 打字速度 | Slider | 中 |
| Auto 速度 | Slider | 中 |
| 跳过已读文本 | Toggle | 关 |
| 事件文本字号 | OptionButton | 标准 |

### 4.4 操作

| 项 | 类型 | 默认 |
|---|---|---|
| 键位提示 | Toggle | 开 |
| 鼠标悬停说明 | Toggle | 开 |
| 快捷键列表 | Button | 查看 |

### 4.5 辅助功能

| 项 | 类型 | 默认 |
|---|---|---|
| 字体大小 | OptionButton | 标准 |
| 减少动画 | Toggle | 关 |
| 高对比 UI | Toggle | 关 |
| 颜色辅助标记 | Toggle | 开 |
| 危险操作强确认 | Toggle | 开 |

---

## 5. 交互规则

- 修改设置后可立即预览，但需点击“应用”保存。
- 点击“返回”时如有未保存更改，提示是否放弃。
- “恢复默认”需二次确认。
- 从游戏内打开设置时，关闭后回到暂停菜单。
- 从主菜单打开设置时，关闭后回到主菜单。

## 7. 数据需求

| 数据 | 来源 | 用途 |
|---|---|---|
| 当前设置 | SettingsService | 初始化控件 |
| 默认设置 | SettingsDefaults | 恢复默认 |
| 进入来源 | NavigationContext | 返回正确界面 |

---

## 8. 验收标准

- [ ] 主菜单和暂停菜单都能打开设置界面。
- [ ] 音频、显示、文本、操作、辅助功能分类可切换。
- [ ] 修改设置后点击应用能保存。
- [ ] 未保存更改时返回会提示确认。
- [ ] 恢复默认前必须二次确认。
- [ ] 减少动画、高对比 UI、字体大小设置存在并可保存。
