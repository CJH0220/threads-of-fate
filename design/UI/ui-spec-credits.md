# UI 规格：关于我们 / Credits

> 文档状态：初稿  
> 更新日期：2026-06-10  
> 所属阶段：P2  
> 引擎方向：Godot  
> 美术方向：简约像素风

---

## 1. 用途

关于我们界面用于展示制作人员、素材鸣谢、工具鸣谢、版本信息和版权说明。

---

## 2. 入口与出口

| 类型 | 说明 |
|---|---|
| 入口 | 主菜单点击“关于我们” |
| 出口 | 返回主菜单 |

---

## 3. 布局规格

```text
┌──────────────────────────────────────┐
│ 关于我们                       [返回] │
├──────────────────────────────────────┤
│ 《命运的织线》                         │
│                                      │
│ 制作：                                │
│ - 策划                                │
│ - 程序                                │
│ - 美术                                │
│ - 音频                                │
│                                      │
│ 鸣谢：                                │
│ - 使用工具                            │
│ - 素材来源                            │
│                                      │
│ 版本：v0.1.0                          │
└──────────────────────────────────────┘
```

---

## 4. 内容模块

| 模块 | 内容 |
|---|---|
| 游戏标题 | 《命运的织线》 |
| 制作人员 | 姓名 / 昵称 / 职责 |
| 特别鸣谢 | 测试、顾问、支持者 |
| 工具鸣谢 | Godot、字体、音频工具等 |
| 素材鸣谢 | 美术、音效、字体素材来源 |
| 版本信息 | 当前版本号 |
| 版权说明 | 可选 |

---

## 5. 交互规则

- 内容超出屏幕时使用滚动。
- Esc 或返回按钮回到主菜单。
- 若包含外部链接，点击前应有确认，P2。
- 不在 MVP 中强制制作复杂滚动演出。

---

## 6. Godot 实现建议

```text
CreditsScreen (Control)
├── PanelContainer
│   ├── Header
│   ├── CreditsScroll (ScrollContainer)
│   │   └── CreditsText (RichTextLabel / VBoxContainer)
│   └── BackButton
```

---

## 7. 数据需求

| 数据 | 来源 | 用途 |
|---|---|---|
| 制作人员列表 | 静态配置 / credits.json | 显示人员 |
| 版本号 | ProjectSettings | 显示版本 |
| 鸣谢信息 | 静态配置 | 显示鸣谢 |

---

## 8. 验收标准

- [ ] 主菜单点击关于我们能进入该界面。
- [ ] 界面显示游戏名、制作人员、鸣谢和版本号。
- [ ] 内容超出时可以滚动。
- [ ] 点击返回或 Esc 能回到主菜单。
- [ ] 关于我们界面不影响已有存档和设置状态。
