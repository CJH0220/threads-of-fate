# UI 规格：人物面板

> 文档状态：初稿  
> 更新日期：2026-06-10  
> 所属阶段：P0  
> 引擎方向：Godot  
> 美术方向：简约像素风

---

## 1. 用途

人物面板用于查看单个 NPC 的身份、当前状态、今日行程、关系缘线和业线进度。它帮助玩家判断这个角色是否值得关注、是否存在风险，以及是否需要干预。

---

## 2. 入口与出口

### 入口

| 来源 | 触发 |
|---|---|
| 主地图 | 点击人物头像 / 标记 |
| 地点面板 | 点击今日人物列表 |
| 事件面板 | 点击参与角色头像 |
| 缘线视图 | 点击关系对象 |
| 业线视图 | 点击角色条目 |

### 出口

| 操作 | 结果 |
|---|---|
| 关闭按钮 | 返回上一层 |
| Esc | 返回上一层 |
| 点击相关地点 | 关闭或跳转地点面板 |
| 点击相关事件 | 跳转事件详情或事件推进面板 |

---

## 3. 布局规格

```text
┌──────────────────────────────────────────────┐
│ 头像 / 姓名 / 身份              [关闭]        │
├──────────────────────────────────────────────┤
│ 简介：2-3 句人物描述                         │
├─────────────────────┬────────────────────────┤
│ 今日行程             │ 当前状态               │
│ 早：学校             │ 心智 6                 │

│ 晚：寺庙             │ 魅力 7                 │
│                     │ 信仰 5                 │
├─────────────────────┴────────────────────────┤
│ 缘线：陈远舟 友情 Lv.3 / 顾沉舟 利益 Lv.2     │
├──────────────────────────────────────────────┤
│ 业线：主业线 [■■■■□□] 65% 当前节点：……       │
└──────────────────────────────────────────────┘
```

## 4. 组件清单

| 组件 | 类型 | 内容 | 交互 |
|---|---|---|---|
| 人物头像 | `TextureRect` | 像素头像 | 无 |
| 姓名身份 | `Label` | 姓名、职业、年龄可选 | 无 |
| 简介区 | `RichTextLabel` | 简短人物描述 | 可滚动 |
| 今日行程 | 列表 | 早/午/晚地点 | 点击地点跳转地点面板 |
| 当前状态 | 图标 + 数值 | 心智、体魄、魅力、信仰 | Hover 解释 |
| 缘线列表 | 列表 | 关系对象、性质、等级 | 点击对象跳转人物面板 |
| 业线摘要 | 进度条 + 文本 | 主业线、支线进度 | 点击查看业线视图 |
| 关闭按钮 | `Button` | 返回 | 关闭面板 |

---

## 5. 信息优先级

1. 姓名、头像、身份
2. 今日行程
3. 当前状态
4. 缘线关系
5. 业线进度
6. 详细简介和历史记忆

MVP 不展示完整 Agent Memory，只显示系统整理后的简短摘要。

---

## 6. 缘线展示

| 字段 | 示例 |
|---|---|
| 对象 | 陈远舟 |
| 关系性质 | 友情 |
| 等级 | Lv.3 |
| 强度 | 65 / 100，可隐藏具体数值 |
| 近期变化 | +5，可选 |

颜色规则：

| 关系 | 颜色 |
|---|---|
| 爱情 | 红 / 粉 |
| 友情 | 蓝 |
| 利益 | 金 |
| 陌生 | 灰 |
| 仇恨 | 黑 / 暗红 |

---

## 7. 业线展示

| 字段 | 内容 |
|---|---|
| 主业线名称 | 如“离开小镇” |
| 当前进度 | 0%–100% |
| 当前节点 | 当前正在推进的关键事件 |
| 风险提示 | 若有危险节点，显示红色警告 |
| 支线数量 | 显示支线入口 |

MVP 表现：进度条 + 当前节点一句话。  
后续表现：业线树或时间线。

---

## 8. 状态与变体

| 状态 | 表现 |
|---|---|
| 正常 | 展示完整人物信息 |
| 未解锁人物 | 头像暗化，显示“尚未了解此人” |
| 无缘线 | 显示“暂无明确关系” |
| 无业线 | 显示“命运尚未显形” |
| 危险状态 | 人物头像或业线区域显示红色警告 |
| 数据缺失 | 显示兜底文本，不阻塞界面 |

## 9. Godot 实现建议

```text
CharacterPanel (PanelContainer)
├── Header (HBoxContainer)
│   ├── Portrait (TextureRect)
│   ├── NameBlock (VBoxContainer)
│   └── CloseButton

├── MiddleContent (HBoxContainer)
│   ├── ScheduleList (VBoxContainer)
│   └── StatsGrid (GridContainer)
├── BondList (VBoxContainer)
└── DestinySummary (VBoxContainer)
```

---

## 10. 数据需求

| 数据 | 来源 | 用途 |
|---|---|---|
| NPC 基础信息 | NPCDatabase | 姓名、头像、身份、简介 |
| 当前属性 | NPCStateSystem | 心智、体魄、魅力、信仰 |
| 今日行程 | NPCScheduleSystem | 早/午/晚地点 |
| 缘线 | BondSystem | 关系列表 |
| 业线 | DestinySystem | 主线与支线进度 |
| 解锁状态 | KnowledgeSystem | 控制隐藏信息 |

---

## 11. 验收标准

- [ ] 点击地图人物后能打开对应人物面板。
- [ ] 人物面板显示头像、姓名、身份和简介。
- [ ] 今日早/午/晚行程显示正确。
- [ ] 当前状态至少显示 4 个核心属性。
- [ ] 缘线列表能显示关系对象、关系类型和强度。
- [ ] 点击缘线对象能跳转到对应人物面板。
- [ ] 业线区域能显示主业线名称、进度和当前节点。
- [ ] 无缘线、无业线、未解锁人物都有明确空状态。
