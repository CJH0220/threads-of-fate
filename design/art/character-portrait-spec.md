# 角色头像与立绘规格

> 文档状态：初版  
> 更新日期：2026-06-13  
> 用途：定义 NPC 像素头像、半身像和 UI 表情差分的制作范围

---

## 1. 总体方向

角色美术采用低成本像素头像 + 少量表情差分，不做 Live2D，不做高成本全身动画。

目标：

- 玩家能快速识别角色。
- 角色气质与剧情线一致。
- UI 面板中可读性优先。
- 表情差分服务剧情状态，不追求数量。

---

## 2. 资产规格

| 资产类型 | 建议尺寸 | 用途 |
|---|---:|---|
| 小头像 | 48x48 | 列表、HUD、事件参与者 |
| 中头像 | 96x96 | 人物面板、地点面板 |
| 半身像 | 160x240 或 192x256 | 事件面板重点角色 |
| 表情差分 | 基于中头像或半身像 | 对话、事件状态 |

P0 可以只做小头像 + 中头像，半身像作为 P1。

## 5. 表情差分规则

P0 每名角色最多 4 个表情：

1. 普通。
2. 正向。
3. 压力 / 负向。
4. 剧情关键。

P1 可先只做普通 + 负向。

表情命名建议：

```text
portrait_lin_chaoyin_normal
portrait_lin_chaoyin_tired
portrait_lin_chaoyin_afraid
portrait_lin_chaoyin_relieved
```

---

## 6. UI 使用规则

| UI | 使用资产 |
|---|---|
| HUD 事件标记 | 小头像 |
| 人物面板 | 中头像 |
| 地点面板 | 小头像列表 |
| 事件面板 | 中头像或半身像 |
| 结局界面 | 中头像 + 状态标签 |

如果头像缺失，使用角色剪影 + 姓名，不阻塞研发。

---

## 7. 色彩与可访问性

- 角色阵营和风险不能只靠颜色表达。
- 高风险角色需配合警示图标或状态文字。
- 小头像需要在 48x48 尺寸下仍可辨认。
- 同校学生可以统一校服，但用发型、姿态和配件区分。

---

## 8. 验收标准

- [ ] P0 5 名角色小头像可相互区分。
- [ ] 林潮音、陈远舟、陈海生有至少 4 个表情方向。
- [ ] 顾沉舟在普通表情下不显得明显邪恶。
- [ ] 慧圆同时体现和善与不可靠感。
- [ ] 头像在人物列表、事件面板和结局界面中都可读。
