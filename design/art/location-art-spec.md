# 地点美术规格

> 文档状态：初版  
> 更新日期：2026-06-13  
> 用途：为像素风地点、地图、背景和 UI 缩略图提供美术制作方向

---

## 1. 总体方向

《命运的织线》的地点美术采用低成本 2D 像素风，重点是清晰识别、温暖日常与暗潮风险并存。

关键词：

```text
归潮镇 / 海岛 / 旧土地庙 / 潮湿空气 / 暖色灯光 / 低成本像素 / 民俗暗示
```

## 3. 通用像素规格

| 项目 | 建议 |
|---|---|
| 基础格 | 16x16 或 32x32 tile |
| 地点插图 | 320x180 或 384x216，可等比放大 |
| UI 缩略图 | 96x64 或 128x72 |
| 角色站位图层 | 前景 / 中景 / 背景分层 |
| 色彩数量 | 单地点主色 3-5 个，强调色 1-2 个 |
| 动画 | P0 只做简单 2-4 帧循环 |

## 5. 主地图表现

主地图建议使用简化俯视图或图标地图。

P0 地图只需：

- 寺庙。
- 学校。
- 咖啡店。
- 海鲜店。
- 港口灰显或锁定。

地点状态用图标和文字共同表达：

| 状态 | 表现 |
|---|---|
| 可访问 | 正常亮度 |
| 当前事件 | 小铃铛或命运线闪烁 |
| 高风险 | 警示图标 + 边框形状，不只用红色 |
| 锁定 | 半透明 + 锁图标 |
| 新线索 | 小纸条图标 |

---

## 6. 验收标准

- [ ] P0 地点能在缩略图尺寸下清楚区分。
- [ ] 危险地点不只依赖颜色提示。
- [ ] 寺庙能承担主视觉锚点。
- [ ] 学校、咖啡店、海鲜店能分别体现青春、日常、家庭压力。
- [ ] 所有地点可先用静态图完成，不依赖复杂动画。
